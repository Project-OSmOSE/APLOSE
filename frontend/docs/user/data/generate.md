# Generate a dataset

::: details Terminology

**Dataset**

<!--@include: ../terminology/dataset.md-->

:::

We developed our Python package to manipulate audio files: [OSEkit](https://github.com/Project-OSmOSE/OSEkit).
This package allows to directly output the dataset in APLOSE expected format.

The datasets must be located in a `datasets` folder.

You can find its documentation here: https://project-osmose.github.io/OSEkit/

::: tip

We recommend using the latest version

:::
