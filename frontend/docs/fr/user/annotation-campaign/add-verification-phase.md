# Ajouter une phase de "Verification"

::: details Terminologie

**Campagne d'annotation - Annotation Campaign**

<!--@include: ../terminology/campaign.md-->

**Phase d'annotation - Annotation Phase**

<!--@include: ../terminology/phase.md-->

:::

Vous devez disposer d'une [phase « Annotation »](add-annotation-phase) pour créer une phase « Vérification ».

Pour créer une phase, cliquez sur l'onglet « Vérification ».

![](/annotation-campaign/add-verification-phase.png)

Ensuite, cliquez sur "Create".
