# Générer un dataset

::: details Terminologie

**Dataset**

<!--@include: ../terminology/dataset.md-->

:::


Nous avons développé notre package Python pour manipuler les fichiers audio : [OSEkit](https://github.com/Project-OSmOSE/OSEkit).
Ce package permet de sortir directement l'ensemble des données dans une structure de fichier qui peut être lue par
APLOSE.

Vous pouvez également accéder directement à sa documentation : https://project-osmose.github.io/OSEkit/

::: tip Recommendation

Nous vous recommandons d'utiliser la dernière version de OSEkit.

:::
